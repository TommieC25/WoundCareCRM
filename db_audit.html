🔍 DB Audit — WoundCare CRM

Finds non-Mohs dermatologists, self-named practices, out-of-territory locations, and orphaned providers

🏷️ Corrupted Specialty Values —

Valid specialties are exactly: Podiatry, Wound Care, Dermatology (Mohs surgeons only), Other, Staff. Anything else must be remapped. Select a target for each bad value and tap Fix All Mapped, or fix groups individually.

🦶 DPM Degree / Specialty Mismatch —

Providers with degree DPM (Doctor of Podiatric Medicine) who do not have specialty = Podiatry. These are always podiatrists — one-click fix updates them all.

🎓 Degree Periods (M.D. → MD) —

Providers whose degree field contains periods (e.g. M.D., D.P.M.). Tap Fix All to strip all periods from every affected degree.

📞 Multi-Number Phone Fields —

Locations where the phone or fax field contains multiple numbers separated by "/" or ",". Each number must be in its own field for one-tap dialing. Fix splits the first number into phone and the second into fax (only if fax is currently empty — won't overwrite existing fax data).

⚠️ Solo-Named Practices (Review Required) —

REVIEW ONLY — do not delete without verifying. These are practices whose name closely matches a provider's own name. Many are legitimate solo practices (e.g. "Adam Plotkin DPM LLC" is Dr. Plotkin's real office). Some may be NPI import duplicates. Check each one in the CRM — only delete if the provider has a real group practice and this is a duplicate with no unique data (notes, staff, hours) of its own.

🔪 Dermatology Providers — Mohs Unconfirmed —

Dermatology in this CRM = Mohs surgeons ONLY. General dermatologists are never targets. This list shows every provider with specialty=Dermatology whose name, title, degree, or notes contain no mention of "Mohs". Review each — delete non-Mohs derms, or add "Mohs" to their notes/title to confirm them.
